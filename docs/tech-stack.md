# 기술 스택

## Front-end

- **Framework**: Flutter 3.7.2+ (Dart 3.0+)
- **Architecture**: MVVM (ChangeNotifier, InheritedWidget)
- **UI**: Material Design, Custom Themes
- **PDF 라이브러리**: pdfrx
- **OCR**: Google ML Kit, Tesseract (pytesseract)
- **State Management**: Provider 패턴
- **Local DB**: Isar (NoSQL)
